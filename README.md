📌 Proyecto: Análisis Exploratorio de Datos (EDA)
📖 Descripción
Este proyecto tiene como objetivo realizar un análisis exploratorio de datos sobre campañas de marketing de una entidad bancaria portuguesa, con el fin de identificar los factores que influyen en la contratación de depósitos a plazo.
________________________________________
🎯 Objetivo
Analizar el comportamiento de los clientes y detectar patrones que expliquen la suscripción del producto bancario.
Herramientas utilizadas
•	Python 
•	Pandas 
•	Matplotlib 
•	Seaborn 
•	Visual Studio Code 
________________________________________
🔄 Proceso realizado
1.	Carga de datos 
o	Dataset CSV (campañas de marketing) 
o	Dataset Excel (datos demográficos) 
2.	Limpieza de datos 
o	Eliminación de duplicados 
o	Conversión de fechas 
o	Transformación de variables 
3.	Unión de datasets 
o	Merge mediante identificador único 
4.	Análisis exploratorio 
o	Estadísticas descriptivas 
o	Agrupaciones
•	•  Correlaciones 
•  Visualización 
•	Histogramas 
•	Boxplots 
•	Heatmap
Principales conclusiones
•	La mayoría de los clientes no contrata el producto, indicando un dataset desbalanceado. 
•	La duración de la llamada es el factor más influyente: a mayor duración, mayor probabilidad de contratación. 
•	Un mayor número de contactos con el cliente reduce la probabilidad de éxito. 
•	Los clientes con mayor edad e ingresos tienden a contratar más. 
•	Existen diferencias en la tasa de conversión según la profesión del cliente. 
________________________________________
________________________________________
📌 Conclusión final
El análisis muestra que la estrategia de marketing debería centrarse en mejorar la calidad de las llamadas en lugar de aumentar su cantidad, así como segmentar a los clientes en función de su perfil demográfico y económico.
